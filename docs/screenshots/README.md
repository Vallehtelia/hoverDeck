# Screenshots

Drop PNGs here for the README and a future website. These are **not** bundled
into the packaged `.exe` (only `assets/` is) — they're docs only.

Suggested names (the root `README.md` references these, so they render once added):

| File | Shows |
|------|-------|
| `deck.png` | The deck in normal use — the hero shot |
| `edit-mode.png` | Edit mode: pencil badges, an empty "+ Add a key" socket |
| `action-editor.png` | The step-chain editor (a couple of steps) |
| `script-manager.png` | Scripts dialog (list + editor + install bar) |
| `settings.png` | Settings → General |
| `peek.png` | Tucked to the edge behind the amber peek lamp |
| `vault.png` | The PIN pad / unlocked vault page (optional) |

Tips: capture at 100% scale on a dark background so the gunmetal housing reads
well; PNG keeps the keycap edges crisp. Keep each under ~500 KB if you can.
